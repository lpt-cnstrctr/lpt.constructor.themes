3 #layout Позиционирование

Особый класс свойств которые устанавливают поведение элементов. Все классы делятся на те, что устанавливаются на контейнеры (nz-layout-...) и те что устанавливаются на элементы (nz-flex-...). Каждый класс имеет вспомогательный суфикс "состояние", с его помощью можно применять классы только при определенных разрешениях экрана или иных условий. Сейчас существует два состояния: xs (экран меньше 600px) или print (макет при печати). Пример:

  • .nz-hide-$(состояние)
  • .nz-hide
  • .nz-hide-print
  • .nz-hide-xs
Example 
<style>
  .demo1 {
    background-color: var(--accent-color);
  }
  .demo2 {
    background-color: var(--danger-color);
  }
  .demo1,
  .demo2 {
    color: #fff;
    border: 1px solid var(--secondary-color);
    line-height: 50px;
    text-align: center;
  }
</style>

3.2 #layout.flex Ширина

Устанавливает ограничение по ширине, применяется к элементу.

Задается как .nz-flex-$(состояние)-$(шаг), где шаг - число от 0 до 100 (с шагом 5) и числа 33, 66, а состояние - xs (экран меньше 600px) или print (макет при печати), без состояния значение применяется в любом состоянии.

Шаг устанавливает ограничение по ширине (или высоте) в процентах.

Дополнительно существуют вспомогательные классы:

  • .nz-flex$(состояние)-grow - занимает всю ширину, аналогичен на nz-flex-100
  • .nz-flex$(состояние)-initial - растягивается по необходимости, минимальный размер согласно контенту
  • .nz-flex$(состояние)-auto - растягивается и ужимается по необходимости, изначально занимает пространство согласно контенту
  • .nz-flex$(состояние)-none - не будет растягиватся или ужиматься, будет занимать пространство согласно контенту
  • .nz-flex$(состояние)-noshrink - растягивается по необходимости, но не сжимается
  • .nz-flex$(состояние)-nogrow - сжимается по необходимости, но не растягивается

3.4 #layout.flex-offset Отступ

Устанавливает отступ, применяется к элементу. Задается как .nz-flex-offset-$(состояние)-$(шаг), где шаг - число от 0 до 95 (с шагом 5) и числа 33, 66, а состояние - xs (экран меньше 600px) или print (макет при печати), без состояния значение применяется в любом состоянии. Отступ задается в зависимости от контейнера. Для .nz-layout-column margin-top, а для .nz-layout-row margin-left

Example

    [class="nz-flex"]
  </div>
</div>

3.5 #layout.hide Скрыть

Скрывает элемент (display: none !important), используется вместе с состоянием.

Задается как .nz-hide-$(состояние), где состояние - xs (экран меньше 600px) или print (макет при печати).

3.6 #layout.padding-margin Отступы и переносы

Вспомогательные классы для контейнеров, каждый класс имеет модификатор "состояние". Состояние - xs (экран меньше 600px) или print (макет при печати). Как nz-layout-padding-print-1 или nz-layout-wrap-xs

  • .nz-layout-padding-1 - небольшой отступ (padding)
  • .nz-layout-padding-2 - отступ (padding)
  • .nz-layout-padding-3 - большой отступ (padding)
  • .nz-layout-margin-1 - небольшое поле (margin)
  • .nz-layout-margin-2 - поле (margin)
  • .nz-layout-margin-3 - большое поле (margin)
  • .nz-layout-wrap - перенос на новую строку, когда заканчивается доступное пространство
  • .nz-layout-nowrap - все элементы будут размещены на одной строке
  • .nz-layout-fill - занимает все доступное пространство
